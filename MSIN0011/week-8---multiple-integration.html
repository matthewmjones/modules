Topic 8 Week 8 - Multiple Integration

In this final week’s work you will learn how to integrate functions \[f\colon\mathbb{R}^2 \to \mathbb{R}.\] The methods can be generalised to any number of variables, but the calculations are quite awkward. So we’ll stick to two variables.

8.1 Introduction to integration

First let’s review some important concepts from integration. First we define the integral to be the area beneath the graph of a function \(f\) between two points \(a\) and \(b\).

The intgral

The notation for this is normally \[\int_a^b f(x) \,\mathrm{d}x,\] however we might also write it as \[\int_{[a,b]} f(x) \,\mathrm{d}x\] to make clear that it is the area under the curve for all values in the closed interval \([a,b]\). In general we can, if we wanted, replace \([a,b]\) with any nice enough set \(E\) and write this as \[\int_E f(x) \,\mathrm{d}x.\]

For functions of two variables we do the same thing. We start with a set \(E\subset \mathbb{R}^2\) and define the integral to be the volume under the surface. In Figure \(\ref{volume under a surface}\) we have drawn a typical example, with the set \(E\) being a square on the \(x,y\)-plane.

We call this a double integral and write it as \[\iint_E f(x,y) \,\mathrm{d}x\mathrm{d}y.\]

The volume under a surface

To integrate the function \(f(x,y) = x^2 y\) over this rectangle we treat each variable separately, reminiscent of when we were calculating partial derivatives. So with \(E = \{ (x,y)\colon 1\le x\le 2,\; 2\le y\le 4\}.\)

\[ \begin{aligned} \iint_E x^2 y \,\mathrm{d}x\mathrm{d}y &= \int_2^4 \int_1^2 x^2 y \,\mathrm{d}x\mathrm{d}y\\ &= \int_2^4 \left(\int_1^2 x^2 y \,\mathrm{d}x\right) \mathrm{d}y\\ &= \int_2^4 \left[ \frac{x^3 y}3 \right]_1^2 \,\mathrm{d}y\\ &= \int_2^4 \frac{7y}3 \,\mathrm{d}y\\ &= \left[ \frac{7y^2}6\right]_2^4\\ &=\frac{84}6 = 14 \end{aligned} \]

Note: when integrating over a rectangle be careful to match the limits on the single integrals with the differential \(\mathrm{d}x\) or \(\mathrm{d}y\).

Note: For a rectangle it does not matter which variable you integrate with respect to first, the answer is the same. Again, reminiscent of the mixed second partial derivatives. This is called Fubini’s Theorem in its full glory and is so far beyond the scope of this course I’d have to spend years teaching all the incredible theory of integration before we even got to the point where you could revel in it.

To do this we need limits for the integral with respect to \(x\) and limits for the integral with respect to \(y\).

The set \(E\) can be described as \[ E = \{ (x,y)\colon 0\le x\le 4,\; 0\le y \le -x(x-4)\}. \] Notice how the \(y\) limits depend on \(x\). This means we will need to integrate with respect to \(y\) first. Therefore, \[ \begin{aligned} \iint_E x^2 y \,\mathrm{d}x\mathrm{d}y &= \int_0^4 \int_0^{-x(x-4)}x^2 y \,\mathrm{d}y\mathrm{d}x\\ &= \int_0^4 \left[ \frac{x^2y^2}2 \right]_0^{-x(x-4)} \mathrm{d}x\\ &= \int_0^4 \frac{1}2 x^4(x-4)^2 \,\mathrm{d}x = \int_0^4 \frac{1}2x^6 -4x^5+8x^4 \,\mathrm{d}x\\ &= \left[\frac{1}{14}x^7 - \frac{2}3 x^6+\frac{8}5x^5\right]_0^4\\ &= \frac{8192}{105}\approx 78.02 \qquad\text{to 2 d.p.} \end{aligned} \]

8.5 Integrals over unbounded sets

It is possible for the set \(E\) over which we are integrating to be unbounded. For example we might have \[ E = \{ (x,y) \colon 0\le x<\infty, \; -1 \le y \le 1 \} \] The work we did dealing with improper integrals in Week 2 can be used here.

8.5.1 Example

With \(E\) above calculate \[\iint_E e^{-(x+y)} \,\mathrm{d}x\mathrm{d}y\] Here we follow the same approach as above,

\[ \begin{aligned} \iint_E e^{-(x+y)} \,\mathrm{d}x\mathrm{d}y &= \int_{-1}^1 \int_0^\infty e^{-(x+y)} \,\mathrm{d}x\mathrm{d}y\\ &= \int_{-1}^1 \lim_{z\to\infty}\left[ -e^{-(x+y)}\right]_0^z \,\mathrm{d}y\\ &= \int_{-1}^1 \lim_{z\to\infty} \left(e^{-y} -e^{-(z+y)}\right) \,\mathrm{d}y\\ &= \int_{-1}^1 e^{-y}\lim_{z\to\infty} \left(1 -e^{-z}\right) \,\mathrm{d}y\\ &= \int_{-1}^1 e^{-y}\times 1 \,\mathrm{d}y= \int_{-1}^1 e^{-y} \,\mathrm{d}y\\ &= \left[ - e^{-y}\right]_{-1}^1\\ &= e-\frac{1}e. \end{aligned} \]

We could try to decribe this in our usual way, however note that if we change to polar coordinates then this can be more simply described as \[ E = \{(r,\theta)\colon 1\le r\le 3, \; 0\le\theta \le \pi/2\}.\]

If we now want to find the integral \[\iint_E x^2 y \,\mathrm{d}x\mathrm{d}y\] we must change from using \((x,y)\) coordinates to \((r,\theta)\). This is called a change of variables and we will learn how to do it in this section.

Recall for single variable integrals if we change from \(x\) to another variable \(u\) using a change of variable \(x = g(u)\), then \[ \int_a^b f(x) \,\mathrm{d}x = \int_{g^{-1}(a)}^{g^{-1}(b)} f(g(u)) \frac{\mathrm{d}x}{\mathrm{d}u} \mathrm{d}u. \] Note: The function \(g\) must have an inverse for us to be able to use this formula.

The effect of changing from \(x\) to \(u\) in this integral introduces a distortion to the value of the integral. Multiplying by the derivative \(\dfrac{\mathrm{d}x}{\mathrm{d}u}\) compensates for this distortion.

When we change the variables in multiple integrals we also have to do the same thing. This time we must multiply by the Jacobian determinant.

Suppose we change from \((x,y)\)-coordinates to \((r,\theta)\)-coordinates (polar coordinates). This is done using the equations

\[ \begin{aligned} x &= r\cos\theta\\ y &= r\sin\theta \end{aligned} \]

The Jacobian matrix is the \(2\times 2\) matrix

\[ \frac{\partial (x,y)}{\partial (r,\theta)} = \begin{pmatrix} \dfrac{\partial x}{\partial r} & \dfrac{\partial x}{\partial \theta} \\[8pt] \dfrac{\partial y}{\partial r} & \dfrac{\partial y}{\partial \theta} \end{pmatrix} \] Here we may calculate this as \[ \frac{\partial (x,y)}{\partial (r,\theta)} = \begin{pmatrix} \cos\theta & -r\sin\theta\\ \sin\theta & r\cos\theta \end{pmatrix} \] In general a matrix, thought of as a transformation from \(\mathbb{R}^2\) to \(\mathbb{R}^2\) maps a square of area \(1\) onto a parallelogram whose area is the determinant of the matrix.

This leads to the formula for changing the variables from \((x,y)\)-coordinates to \((r,\theta)\)-coordinates is \[ \iint_E f(x,y) \,\mathrm{d}x\mathrm{d}y = \iint_E f(r\cos\theta, r\sin\theta) \left|\det \frac{\partial (x,y)}{\partial (r,\theta)} \right| \,\mathrm{d}r\mathrm{d}\theta \] Now \[ \det \frac{\partial (x,y)}{\partial (r,\theta)} = \det \begin{pmatrix} \cos\theta & -r\sin\theta\\ \sin\theta & r\cos\theta \end{pmatrix} = r(\cos^2\theta + \sin^2\theta) = r. \] Hence the following formula holds, \[ \iint_E f(x,y) \,\mathrm{d}x\mathrm{d}y = \iint_{E'} f(r\cos\theta, r\sin\theta) \,r\mathrm{d}r\mathrm{d}\theta \] where we write \(E'\) to be the set \(E\) but in polar coordinates.

Let us calculate the integral of \(x^2y\) over the region \(E\), where \[ E = \{(r,\theta)\colon 1\le r\le 3, \; 0\le\theta \le \pi/2\}.\] Therefore, \[ \begin{aligned} \iint_E x^2y \,\mathrm{d}x\mathrm{d}y &= \int_0^{\pi/2}\int_1^3 (r\cos\theta)^2(r\sin\theta) \,r\,\mathrm{d}r\mathrm{d}\theta\\ &= \int_0^{\pi/2}\int_1^3 r^4 \cos^2\theta\sin\theta \,\mathrm{d}r\mathrm{d}\theta\\ &= \int_0^{\pi/2} \left[\frac{r^5}5 \cos^2\theta\sin\theta \right]_1^3 \mathrm{d}\theta\\ &= \frac{242}5\int_0^{\pi/2} \cos^2\theta\sin\theta \,\mathrm{d}\theta\\ &= \frac{242}5 \left[ -\frac{1}3 \cos^3 \theta\right]_0^{\pi/2}\\ &= \frac{242}{15} \end{aligned} \]

8.7 Changing variables in general

Suppose we can write the coordinates \((x,y)\) as functions of a second coordinate system, \((u,v)\), as follows. \[ (x,y) = (f_1(u,v), f_2(u,v)).\] Suppose also that \(E\) can be described in the new coordinate system as the set \(F\). Then we have the change of variable formula:

\[ \iint_E f(x,y) \,\mathrm{d}x\mathrm{d}y = \iint_F f(u, v) \left|\det \frac{\partial (x,y)}{\partial (u,v)} \right| \,\mathrm{d}u\mathrm{d}v \] where \[ \frac{\partial (x,y)}{\partial (u,v)} = \begin{pmatrix} \dfrac{\partial f_1}{\partial u} & \dfrac{\partial f_1}{\partial v} \\[8pt] \dfrac{\partial f_2}{\partial u} & \dfrac{\partial f_2}{\partial v} \end{pmatrix} \]

Points to note about this:

  • You must write \(x\) and \(y\) as functions of \((u,v)\)
  • The function that maps \((u,v)\) to \((x,y)\) must be invertible

If we look at the equations of the lines bounding the region then we can write them as \[y + 2x = 2, \quad y + 2x = 3, \quad y-x = 0, \quad y-x = 2.\] This suggests a useful change of variable would be \[ u = y + 2x, \qquad v = y - x.\] Then in the \((u,v)\) coordinate system, \(E\) becomes the rectangle, \[F = \{ (u,v)\colon 2\le u\le 3, \; 0\le v\le 2\}.\] Now, we need to write \(x\) and \(y\) as functions of \((u,v)\). Write \[ \begin{aligned} u &= y + 2x\\ v &= y-x \end{aligned} \] Subtracting the second from the first equation we get \[u - v = 3x\] so that \(x = (u - v)/3\). By the second equation, \[y = v + x = v + \frac{u-v}3 = \frac{u+2v}{3}.\] So that our change of variables is \[ x = \frac{u-v}3, \qquad y =\frac{u+2v}{3}.\] Next the Jacobian, \[ \begin{aligned} \frac{\partial (x,y)}{\partial (u,v)} &= \begin{pmatrix} \dfrac{\partial f_1}{\partial u} & \dfrac{\partial f_1}{\partial v} \\[8pt] \dfrac{\partial f_2}{\partial u} & \dfrac{\partial f_2}{\partial v} \end{pmatrix}\\ &= \begin{pmatrix} 1/3 & -1/3\\1/3 & 2/3 \end{pmatrix} \end{aligned} \] And so its determinant is \[\det \begin{pmatrix} 1/3 & -1/3\\1/3 & 2/3 \end{pmatrix} = \frac{1}3.\] Finally, \[x y^2 = \frac{(u-v)(u+2v)^2}{27} = \frac{u^3+3 u^2 v-4 v^3}{27}\] Hence, brining this all together, \[ \begin{aligned} \iint_E x y^2 \,\mathrm{d}x\mathrm{d}y &= \iint_F \frac{1}{27}(u^3+3 u^2 v-4 v^3) \; \frac{1}3 \mathrm{d}u\mathrm{d}v\\ &= \frac{1}{81} \int_0^2 \int_2^3 u^3+3 u^2 v-4 v^3 \,\mathrm{d}u\mathrm{d}v\\ &= \frac{1}{81} \int_0^2 \left[ \frac{1}4u^4 + u^3v - 4v^3u\right]_2^3 \mathrm{d}v\\ &= \frac{1}{81} \int_0^2 \frac{65}4+19v - 4v^3 \,\mathrm{d}v\\ &= \frac{1}{81} \left[ \frac{65}4 v + \frac{19}2v^2 - v^4 \right]_0^2\\ &= \frac{1}{81} \left( \frac{65}2 + 38 - 16 \right)\\ &= \frac{109}{162} \approx 0.673 \qquad\text{to 3 s.f.} \end{aligned} \]

We may write the bounding curves as \[\frac{y}{x^2} = 1, \quad \frac{y}{x^2}=2, \quad y + x^2 = 4, \quad y + x^2=6.\] This suggests the change of variables, \[ u = \frac{y}{x^2}, \qquad v = y + x^2.\] Then in the \((u,v)\) coordinate system \(E\) becomes, \[F = \{ (u,v)\colon 1\le u\le 2, \; 4\le v\le 6\}.\] Now write \(x\) and \(y\) as functions of \((u,v)\).

First we have that \(x^2 = y/u\) so that \[v = y + \frac{y}u = y\left(1 + \frac{1}u\right)\] Hence \[ y = \frac{uv}{1+u} \] Therefore \[ x^2 = \frac{y}u = \frac{v}{1+u}\] and we have \[ x = \left(\frac{v}{1+u}\right)^{1/2}.\] Now \[\frac{\partial x}{\partial u} = -\frac{v^{1/2}}{2(u+1)^{3/2}},\qquad \frac{\partial x}{\partial v} = \frac{1}{2\sqrt{v(1+u)}}\] and

\[\frac{\partial y}{\partial u} = \frac{v}{(u+1)^2}, \qquad \frac{\partial y}{\partial v} = \frac{u}{u+1}\]

So the determinant of the Jacobian matrix is \[ -\frac{v^{1/2}}{2(u+1)^{3/2}} \frac{u}{u+1} -\frac{v}{(u+1)^2} \frac{1}{2\sqrt{v(1+u)}} = -\frac{\sqrt{v}}{2(u+1)^{3/2}}\] Now \[ \begin{aligned} \frac{y}x &= \frac{uv}{1+u}\big/\left(\frac{v}{1+u}\right)^{1/2} \\ &= u\sqrt{v(1+u)} \end{aligned} \] Putting all this together \[ \begin{aligned} \iint_E \frac{y}x \,\mathrm{d}x\mathrm{d}y &= \iint_F u\sqrt{v(1+u)} \, \left| -\frac{\sqrt{v}}{2(u+1)^{3/2}} \right|\,\mathrm{d}u\mathrm{d}v\\ &= \int_4^6 \int_1^2 \frac{u v}{1+u} \,\mathrm{d}u\mathrm{d}v\\ &= \int_4^6 \int_1^2 \frac{v(1+u) - v }{1+u}\,\mathrm{d}u\mathrm{d}v\\ &= \int_4^6 \int_1^2 v - \frac{v}{1+u} \,\mathrm{d}u\mathrm{d}v\\ &= \int_4^6 \left[ u v - v\log (1+u) \right]_1^2 \,\mathrm{d}v\\ &= \int_4^6 v\left(1 - \log \frac{3}2\right) \,\mathrm{d}v\\ &= \left(1 - \log \frac{3}2\right) \left[\frac{v^2}2\right]_4^6\\ &= 10 \left(1 - \log \frac{3}2\right) \approx 5.95 \qquad\text{to 3 s.f.} \end{aligned} \]

8.8 Extra: Changing the order of integration

(This section is not assessed) Consider the example where \(E\) was the set given in Figure \(\ref{example area between axes and curve integral}\). This is the set we described as \[ E = \{ (x,y)\colon 0\le x\le 4,\; 0\le y\le -x(x-4)\}\] We previously integrated with respect to \(y\) first.

Suppose we instead want to integrate with respect to \(x\) first. Then we need to change the way we have described the set. We need to now have a fixed range for \(y\) and a changing range for \(x\) that may depend on \(y\).

To do this first we look at Figure \(\ref{example area between axes and curve integral}\) and note that the range of \(y\) values is between \(0\) and \(4\).

To find the range for \(x\) we consider a fixed \(y\) and think of the horizontal line that intersects the set \(E\). This intersects the curve \(y=-x(x-4)\) in two places. These will be the lower and upper limits for \(x\).

To calculate this we need to solve \[ y = -x(x-4) \] for \(x\). This is a quadratic equation which, when expanded and written in standard form, is \[ x^2 - 4x + y = 0\] The two solutions are \[ x = \frac{4\pm \sqrt{16-4y}}2 = 2\pm \sqrt{4-y}.\] Therefore we have shown that we can describe \(E\) as \[ E = \{(x,y)\colon 0\le y\le 4,\; 2 - \sqrt{4-y} \le x \le 2+\sqrt{4-x} \}.\]

Therefore we can calculate the integral as follows. \[ \begin{aligned} \iint_E x^2 y \,\mathrm{d}x\mathrm{d}y&= \int_0^4 \int_{2-\sqrt{4-y}}^{2+\sqrt{4-y}} x^2 y \,\mathrm{d}x\mathrm{d}y \\ &= \int_0^4 \left[\frac{x^3 y}3 \right]_{2-\sqrt{4-y}}^{2+\sqrt{4-y}} \mathrm{d}y \\ &= \int_0^4 \frac{y}3\left((2+\sqrt{4-y})^3-(2-\sqrt{4-y})^3\right) \mathrm{d}y\\ &= \int_0^4 \frac{2}3 y(4-y)^{1/2} (16-y) \,\mathrm{d}y \\ \end{aligned} \] If we use the substitution \(u = 4 - y\) then \(\mathrm{d}y = -\mathrm{d}u\). Also when \(y=0\), \(u=4\), and when \(y=4\), \(u=0\) (i.e. it swaps the limits around). Then, continuing, \[ \begin{aligned} \iint_E x^2 y \,\mathrm{d}x\mathrm{d}y&= \int_0^4 \frac{2}3(4-u)u^{1/2} (12+u) \,\mathrm{d}u\\ &= \int_0^4 32u^{1/2} - \frac{16}3 u^{3/2} - \frac{2}3 u^{5/2} \,\mathrm{d}u\\ &= \left[ \frac{64 u^{3/2}}{3}-\frac{32 u^{5/2}}{15}-\frac{4 u^{7/2}}{21} \right]_0^4\\ &= \frac{8192}{105} \end{aligned} \] which is the same answer we calculated before.
