Topic 1 Foundations

1.1 Sets

A set is a collection of elements. To describe a set we either list all its element (using ellipses for infinite sets), or use a predicate. For example:

  • \(C = \{ \text{red}, \text{orange}, \text{yellow} \}\)
  • \(\mathbb{N} = \{1, 2, 3, 4, \ldots \}\)
  • \(P = \{ p \colon p \text{ is a prime number}\}\)

To indicate an element is contained in a set we use the symbol \(\in\). For example, \[ \text{orange} \in C.\] Similarly if an item is not in a set we use the symbol \(\not\in\). For example, \[\text{blue} \not\in C.\]

1.1.1 Subsets

If all the elements of a set are also elements of another set then we say the first set is a subset of the second. This is written, for example, \[P \subset \mathbb{N}.\]

1.1.2 Unions and intersections

The union of two sets \(A\) and \(B\) is the set

\[A \cup B = \{ x\colon x \in A\text{ or } x\in B\}.\]

The intersection of two sets \(A\) and \(B\) is the set \[A \cap B = \{ x\colon x \in A\text{ and } x\in B\}.\]

1.1.3 The empty set

The set that contains no element is called the empty set and is denoted by the special symbol \[\emptyset\] The empty set is a subset for every set. In notation, for any set \(A\),

\[\emptyset \subset A.\]

1.2 Intervals

Intervals are especially useful sets of real numbers that lie between two values. The notation for intervals is the following:

\[ \begin{aligned}[] [a, b] &= \{ x\colon a\le x \le b\} \\ (a, b) &= \{ x\colon a< x < b\}, \\ (a, b] &= \{ x\colon a< x \le b\},\\ [a, b) &= \{ x\colon a\le x < b\}. \end{aligned} \]

  • The values \(a\) and \(b\) are called the endpoints
  • Intervals of the form \([a,b]\) are called closed intervals
  • Intervals of the form \((a,b)\) are called open intervals

Note: the values \(a\) and \(b\) must satisfy \(a\le b\) otherwise the interval is the empty set. Also \(a\) or \(b\) or both can be \(\pm \infty\).

For example, \[\mathbb{R} = (-\infty, \infty)\]

1.3 Special sets

There are a number of special sets that we will routinely use so they have their own special characters:

  • \(\mathbb{N} = \{ 1, 2, 3, 4, \ldots \}\), the natural numbers;
  • \(\mathbb{Z} = \{ \ldots, -2, -1, 0, 1, 2, \ldots \}\), the integers;
  • \(\mathbb{Q} = \{ p/q \colon p, q \in \mathbb{Z}, q \neq 0\}\), the rationals; and
  • \(\mathbb{R}\) the real numbers.

You might also have encountered

  • \(\mathbb{C} = \{ a + \mathrm{i}b\colon a, b \in \mathbb{R}\}\), the complex numbers.

Note: we have the following chain of subsets,

\[ \mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}.\]

Note: there is also a bigger set of numbers called Hamiltonians that include the complex numbers as a subset, they’re useful when manipulating 3d vectors in computer graphics. They are denoted \(\mathbb{H}\).

1.4 Tuples

A tuple is an ordered pair of items, for example \((1,2)\). If \(A\) and \(B\) are sets then the set of tuples \((a,b)\) where \(a\in A\) and \(b\in B\) is denoted \[ A \times B. \] Hence, \[ A \times B = \{ (a,b) \colon a\in A, \text{and } b\in B \}.\]

For example, \[ [0,1] \times [-1,1] = \{ (x,y)\colon 0 \le x \le 1, -1 \le y \le 1\}.\] As a special case we write \(A\times A = A^2\). For example,

\[\mathbb{R}^2 = \{ (x,y)\colon x, y \in\mathbb{R}\}.\]

Tuples can be extended to an arbittary number of items, for example,

\[ (3, -2, 1) \] is a \(3\)-tuple and in general an ordered list of \(n\) items is called a \(n\)-tuple.

The definitions above extend to these higher order tuples. For example,

\[[0,2]\times [-1,1]\times [2,3] = \{ (x,y,z)\colon x\in [0,2], y\in [-1,1], z\in [2,3]\}\]

Hence \((1, 0, 2) \in [0,2]\times [-1,1]\times [2,3]\), for example.

The set \(\mathbb{R}^n\) is an important case, it consists of all the \(n\)-tuples of the form \[ (x_1, x_2, \ldots, x_n)\] where each \(x_i\) is a real number.

1.5 Functions

A function \(f\) is a rule that associates to each element of a set \(A\) - called the domain - an element of a second set \(B\) - called the co-domain. We write this as

\[f\colon A \to B\]

The range of the function \(f\) is the set of all elements of \(B\) that are mapped by \(f\) from some element of \(A\). We write this as

\[ f(A). \] For example suppose \(f\colon \mathbb{R}\to \mathbb{R}\) is defined by

\[ f(x) = x^2.\]

Then \(f(\mathbb{R}) = [0,\infty)\).

Figure 1.1: The graph of \(f(x) = x^2\)

The graph of this function is given in Figure \(\ref{fig:squarefunction}\).

Note: the graph of a function \(f\colon A \to B\) is a subset of \(A\times B\).

We often define a function using the following two-line format, where the special arrow \(\mapsto\) is used to show that a variable is mapped to and expression.

\[ \begin{aligned} f\colon &A \to B \\ &x \mapsto f(x) \end{aligned} \]

For example,

\[ \begin{aligned} f \colon &\mathbb{N} \to \mathbb{N}\\ &x \mapsto 3x + 1 \end{aligned} \]

Sometimes a function can be defined with different expressions in different subsets of its domain. In this case we use the following format.

\[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto \begin{cases} 1, & \text{if } x < 1;\\ x^2, & \text{if } 1 \le x \le 2\\ 4, & \text{if } x > 2. \end{cases} \end{aligned} \]

This is called a piecewise function. It graph is given in Figure \(\ref{piecewise function}\).

Figure 1.2: The graph of a piecewise function

Note: the range of this function is \(f(\mathbb{R}) = [1,4]\).

The domain and range of a function need not be one of the nice sets we’ve seen so far. Let \(X = (-1,1) \cup (2\pi,\infty)\). Then we may define,

\[ \begin{aligned} g\colon &X\to \mathbb{R}\\ &x \mapsto \begin{cases} x^2, & \text{if } x \in (-1,1);\\ \sin x, & \text{if } x\in (2\pi,\infty)\\ \end{cases} \end{aligned} \]

This has graph given below in Figure \(\ref{disconnected domain example}\)

Figure 1.3: A domain that isn’t a single interval

Here, the range is \(f(X) = [-1,1]\).

1.6 Function composition

We can compose two funcions \(f\) and \(g\) to produce a third, \(f\circ g\) which is defined to be \[ f\circ g(x) = f( g(x) ),\] for all \(x\). But we do need to be careful. This is only possible if the range of \(g\) is a subset of the domain of \(f\).

For example, suppose \[ \begin{aligned} f\colon & [0,\infty) \to \mathbb{R}\\ & x \mapsto \sqrt{x} \end{aligned} \] and \[ \begin{aligned} g\colon & \mathbb{R} \to \mathbb{R}\\ & x \mapsto \mathrm{e}^x. \end{aligned} \] Then \(f\circ g\) is defined for all \(x\in \mathbb{R}\) since the range of \(g\) satisfies \[ g(\mathbb{R}) = (0,\infty) \subset [0,\infty)\] which is the domain of \(f\).

On the other hand if we define, \[ \begin{aligned} h\colon & \mathbb{R} \to \mathbb{R}\\ & x \mapsto 1-x^2. \end{aligned} \] then \(f\circ h\) cannot be defined because the range of \(h\) is \[h(\mathbb{R}) = (-\infty,1] \not\subset [0,\infty)\]

1.7 Injectivity

A function \(f\) is said to be injective or one-to-one if whenever \(x\neq y\), \(f(x) \neq f(y)\).

The following useful graphical technique can be used to determine if a function is injective:

  • \(f\) is injective if, and only if every horizontal line intersects the graph of \(f\) at at most one point (i.e. either one point or no points)

For example, the function

\[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto 1 - x - 2 x^3 \end{aligned} \] is injective. Its graph is in Figure \(\ref{injective function example}\).

An injective function

Whereas the function

\[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto 1 + x - 2 x^3 \end{aligned} \] is not injective. Its graph is given here, where the horizontal line drawn is an example of a line that intersects the graph at three points.

Figure 1.4: A non-injective function

1.8 Surjectivity

A function \(f\colon A \to B\) is said to be surjective or onto if \[f(A) = B.\] In other words if every element of \(B\) is mapped by some element of \(A\).

For example, \[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto x^2 \end{aligned} \] is not surjective since \(f(A) = [0,\infty) \neq \mathbb{R}\).

Note:

  • surjectivity and injectivity have no relation, a function can be neither, either, or both.
  • We always have the option of making a function \(f\colon A \to B\) surjective by simply defining the co-domain \(B\) to be the range \(f(A)\).

For example if we redefine the above function as

\[ \begin{aligned} f\colon &\mathbb{R}\to [0,\infty)\\ &x \mapsto x^2 \end{aligned} \] then this is surjective.

1.9 Bijectivity and inverses

A function \(f\) is said to be bijective if it is both injective and surjective.

A function \(f\colon A \to B\) is said to be invertible with inverse \(g\colon B \to A\) if for all \(x\in A\) \[ g\circ f(x) = x. \] We normally write the inverse function g as \(f^{-1}\).

A function \(f\) has an inverse if and only if it is bijective.

A useful technique for finding the inverse of a bijective function is to introduce a new variable, \(y\) say and change the subject of the equation \(y = f(x)\). For example, to find the inverse of \(f(x) = x^3 + 1\), \[ \begin{aligned} y &= x^3 + 1\\ y -1 &= x^3 \\ (y-1)^{1/3} &= x \end{aligned} \] So that \(f^{-1} (y) = (y-1)^{1/3}\).

1.10 Increasing and decreasing functions

A function \(f\colon A \to \mathbb{R}\), where \(A \subset\mathbb{R}\) defined to be:

  • inceasing if whenever \(a > b\), we have that \(f(a) \ge f(b)\);
  • strictly increasing if whenever \(a > b\), we have that \(f(a) > f(b)\);
  • decreasing if whenever \(a > b\), we have that \(f(a) \le f(b)\);
  • strictly decreasing if whenever \(a > b\), we have that \(f(a) < f(b)\).

A function that satifies any of the conditions above is said to be monotone.

A function that is either stricly increasing or strictly decreasing is said to be strictly monotone.

1.11 Continuous functions - the basic idea

A function such as \[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto \begin{cases} x^2, & \text{if } x < 2;\\ 4, & \text{if } x \ge 2 \end{cases} \end{aligned} \] is said to be continuous since its graph consists of a single connected curve (see Figure \(\ref{continuous function example}\)).

The graph of a continuous functio

If the graph of a function is not a single connected curve then the function is said to be discontinuous. For example, \[ \begin{aligned} f\colon &\mathbb{R}\to \mathbb{R}\\ &x \mapsto \begin{cases} x^2, & \text{if } x < 2;\\ 2, & \text{if } x \ge 2 \end{cases} \end{aligned} \] is a discontinuous function (Figure \(\ref{disconnected curve example}\)).

1.12 Limits

To understand continuity better we need to understand limits.

The function \(f\colon [0,2] \to \mathbb{R}\) is defined to be \[ f(x) = \begin{cases} x^3, & x\neq 1\\ 2, & x = 1\end{cases} \]

The graph of this is given in Figure \(\ref{limit example}\).

The limit of a function

If \(x\) approaches \(1\) from either side, \(f(x)\) approaches \(1\) also (even through \(f(1) = 2\)). We say that the limit of \(f(x)\) as \(x\) tends to \(1\). This is written as \[\lim_{x\to 1} f(x) = 1.\] In general we can find the limit of a function at any point and this is written \[\lim_{x\to a} f(x)\] and read the limit of \(f(x)\) as \(x\) tends to \(a\).

The idea of the limit is that the values \(f(x)\) should be getting closer and closer to the limit as \(x\) gets closer and closer to \(a\) - although we don’t need \(f(a)\) to be defined or to be the same as the limit.

1.13 Applications of the limit: continuity

A function \(f\colon A \to\mathbb{R}\) is said to be continuous at \(a\) if \(a\) is in the domain of \(f\) and \[ f(a) = \lim_{x\to a} f(x).\] A function is said to be continuous if it is continuous at all points in its domain.

For example, the function we previously defined, \[ f(x) = \begin{cases} x^3, & x\neq 1\\ 2, & x = 1\end{cases} \] \[ \lim_{x\to 1} f(x) = 1 \neq f(1).\]

1.14 Working with limits

To work with limits we introduce a number of important facts about them.

1.14.1 Theorem 1

If \(L = \lim_{x\to a} f(x)\) then \(L\) is unique - i.e. it is not possible for a function to tend to two or more different limits.

1.14.2 Theorem 2

If \(L = \lim_{x\to a} f(x)\) and \(M = \lim_{x\to a} g(x)\) then:

  1. \(\lim_{x \to a} f(x) + g(x) = L + M\)
  2. \(\lim_{x \to a} f(x)g(x) = LM\)
  3. \(\lim_{x \to a} f(x)/g(x) = L/M\) as long as \(M\neq 0\)

For example, since \[ \begin{aligned} \lim_{x \to 2} x^2 &= 4\\ \lim_{x \to 2} \cos \pi x &= 1 \end{aligned} \] we have that \[ \lim_{x\to 2} x^2 \cos \pi x = 4\times 1 = 4. \] We can also say that \[ \lim_{x\to 2} \frac{x^2}{\cos \pi x} = \frac{4}1 = 4. \] And so on.

1.14.3 Theorem 3

The following families of functions are continuous:

  • Polynomials
  • Exponential functions and logarithms
  • The trigonometric functions \(\sin\), \(\cos\) and \(\tan\)
  • The hyperbolic trigonometric functions \(\sinh\), \(\cosh\) and \(\tanh\)
  • Any arithmetic combination of these, or any composition of functions including these

1.15 When limits don’t exist

It’s possible for a function not to have a limit. For example, define, \[ f(x) = \begin{cases} -1, & x < 0;\\ 1, & x \ge 0. \end{cases} \] The graph of this function shown in Figure \(\ref{discontinuous example}\).

Grpah of a function without a limit

As \(x\) approaches \(0\) from the negative side (the left-hand side) \(f(x)\) tends to \(-1\).

On the other hand as \(x\) approaches \(0\) from the positive side (the right-hand side) \(f(x)\) tends to \(1\).

This is an example where the limit does not exist at the point \(a = 0\).
